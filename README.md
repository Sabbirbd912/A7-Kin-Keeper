🌿 KeenKeeper - Personal Connection Tracker
KeenKeeper is a relationship management tool designed to help you nurture and maintain meaningful connections with friends, family, and colleagues. It helps you track when you last interacted with someone and reminds you who needs your attention most.

🚀 Key Features:

📊 Dynamic Dashboard: View real-time stats including total friends, those "On Track," and those who "Need Attention".

🕒 Interaction Timeline: Keep a history of every interaction (Call, Text, Video) with a timestamped log.

🔍 Smart Filtering: Easily filter your timeline entries by interaction type to find specific history.

📈 Friendship Analytics: Visual representation of your communication habits using interactive Pie Charts.

📱 Responsive UI: Fully responsive design built with Tailwind CSS and DaisyUI, optimized for both mobile and desktop.
